Dataset Information
Overview
The dataset used in this project comprises 1,000 synthetic learner profiles designed to simulate real-world learning behavior in Data Structures and Algorithms (DSA). It serves as the foundation for generating personalized learning path recommendations based on learner performance.

Dataset Access
The complete dataset is publicly available on Kaggle:

https://www.kaggle.com/datasets/rithanya06/dsa-learning-path-recommendation-dataset/data

Dataset Description
Each record in the dataset represents an individual learner's interaction with a specific topic and includes the following attributes:

Student_ID – Unique identifier for each learner
Current_Topic – The DSA topic currently being studied
Level – Difficulty level associated with the topic
Score – Assessment score obtained by the learner (out of 10)
Status – Performance classification (Advance / Repeat / Remedial)
Recommended_Next_Topic – Suggested next topic based on performance
Purpose of the Dataset
This dataset is used to:

Model learner performance patterns
Enable personalized learning path generation
Support recommendation logic using rule-based and similarity-based approaches
Note
The dataset is hosted on Kaggle to ensure open access, reproducibility, and compliance with academic standards.
